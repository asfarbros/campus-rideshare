### Project Description

A full-stack web application designed for campus students to share rides, reducing travel costs and improving commute convenience. The app allows users to register, log in, and manage ride details through a secure interface.

### Tech Stack

* *Frontend*: React.js with Vite
* *Backend*: Node.js and Express.js
* *Database*: MongoDB Atlas
* *Deployment*: Vercel (Frontend) and Render (Backend)

---

### Features

* User Authentication (JWT based login and signup)
* Responsive UI for mobile and desktop
* Secure connection to cloud database
* API integration with a live Node.js server
